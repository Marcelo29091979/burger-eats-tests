# 🧪 Projeto de Testes Manuais - Burger Eats

Projeto de testes manuais realizado na aplicação Burger Eats, com foco na validação de regras de negócio, identificação de falhas e garantia da qualidade do sistema.

Foram aplicados testes funcionais, exploratórios e negativos, com documentação completa de casos de teste, execução e bugs com evidências.

🔗 Projeto: https://buger-eats.vercel.app

## 🛠 Tecnologias e Ferramentas

- Testes Manuais
- BDD (Gherkin)
- Postman (API)
- Git e GitHub
- Metodologias Ágeis (Scrum/Kanban)

## 📌 Objetivo
Executar testes manuais baseados em requisitos funcionais de uma aplicação web de cadastro de entregadores.

## 🔍 Escopo
- Tela inicial
- Cadastro de entregadores

## 🧠 Técnicas utilizadas
- Testes manuais
- Testes exploratórios
- Testes negativos
- BDD (Gherkin)

## 🛠 Ferramentas
- Git
- GitHub
- Markdown

## 📌 Conclusão

Durante os testes, foram identificadas falhas relacionadas à validação de dados, como CPF e email inválidos sendo aceitos pelo sistema.

Os testes demonstram a importância da validação de entradas e reforçam o papel do QA na garantia da qualidade do software.
